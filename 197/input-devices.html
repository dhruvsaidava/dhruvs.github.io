Some Quick Points

  • Check Dimension of Board Before Milling ( I try to used board which are left over so we can save material)
  • After Milling Cross Check with PNG file ( Sometime few lines are connected which can be seprated by Cutter use gloves for safety)
  • L shape Header Pins connection can touch the board (I faced that issue so i had to remove headers clean PCB under them with cutter and again resolder it)
