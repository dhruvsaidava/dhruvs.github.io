I am going to make a simple holder for Ultrasonic Sensor which have Neodymium Magnet at base so it can easily fixed on any magnetic structure.

Ultrasonic Sensor Magnetic holder

  1. You need dimensions of Ultrasonic sensor to start with designing a CAD Model
  2. I googled ultrasonic sensor dimensions
  3. You will get such a image with Dimension or you have to use Vernier
  4. Now you have each dimension so now start with 2d and converting that to 3d.

Using Fusion 360 for Ultrasonic holder

  1. Open Parametric Modify > Change Parametric > Add ( Click on Green Plus)

  2. Add Dimension to Parametric So you can directly add.

  3. Now go to Dashboard and Go to Sketch > Line ( Put a point stretch and enter "endtoend" it will automatically take Dimension 41.7 as it is in Parametric
  4. just go Sketch > select Circle >with 2 point
  5. Put on end of line and stretch and enter value diameter it take 16.3 automatically

  6. Now Make it 3D create Rectangle and extrude ( I have Video to explain this check it below)
