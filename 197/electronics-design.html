Tools Used
  • Eagle 8.0.2 | Download (Education Version) | 98 MB
  1. Install Eagle
  2. Add components connect circuit.
  3. after connecting each component and removing Errors go Boards
  4. After that Components to arrange and check for routing
  5. After that evaluate circuit layout and check if wire is not connect in layer ( After trying i found one was still not connecting I am going to use jumper wire
  6. After that go for export of that Circuit
  7. Export in Monochrome and 400 DPI Resolution (Its Important )
To Add components click on add and select according to company

There are different layer to show while making connection for circuit which guide better to make connection properly.

After that you have switch connection to Board were you will find layout of each component is added and you can move according to it
Try to make is as good and easy so routing will be that much accurate.

If you still find it is to be adjusted and few thing you can do is connect "0" resistor to jump any single line

Error Can be check and removed after connection of components

In Routing you have to do some edit in NET Classes from DRU file you are using.
We are using FAB DRU and change that value to mils. ( In Eagle it directly takes as mm).
After that keep top as Auto and bottom as N/A ( I forget to do this in starting)

After readjusting components you will achieve near 100% ( I got 96 % it show blue line on pressing evalute so that can be solved by jumper wire)

After Check for DRU and Spacing check

Finaly you can export.
Files > Export > (Setting Monochrome & 400 dpi)

  • Export in 400 dpi and monochrome
  • Select Components according to Company so dimension would be according to that
